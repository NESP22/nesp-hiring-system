# Craig Quick Reference

## Daily Path

1. **NESP Hiring** -> **Needs Craig**
2. Review applicant.
3. Click **Invite to Schedule Phone Screen** when approved.
4. Copy invitation.
5. Send manually.
6. Check **Phone Screens**.
7. Review completed calls.
8. Decide next human step.

## Safe Calling Rules

- Scheduling links do not call immediately.
- Calls happen only at the candidate-selected time.
- Calls require `NESP_VAPI_ENABLED=1`.
- Audio recording must remain off.
- Transcription happens only after consent.
- No automatic rejection.
- No automatic stage change.

## Emergency Stop

Set `NESP_VAPI_ENABLED=0`, then check scheduler logs.

## Job Ads

- Use **Job Ads** for drafts, tracking links, and source reporting.
- Do not publish, boost, pay, or enter billing without approval.
- Use tracked links like `nesp_source=indeed`.
- Record manual spend only after Craig approves and spends it.

## Never Change Alone

- Render variables
- Vapi assistant settings
- webhook secrets
- migrations
- scheduler jobs
- DNS
- paid ad billing
- applicant automation
